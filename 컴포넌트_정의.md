# 컴포넌트 종류 및 정의

---

### 공통 규칙
- 모든 컴포넌트는 추가/삭제/중복 사용 가능
- 각 컴포넌트별 **Basic / Advanced** 토글로 편집 모드 전환
- 모든 입력란에 **복사/붙여넣기 버튼** 제공
- 삭제된 컴포넌트의 입력값은 localStorage에 보관, 재추가 시 복원
- `초기` 표시가 있는 컴포넌트는 새 청첩장 생성 시 자동 등록됨

---

## 인사말 컴포넌트 `초기`

### 설명
청첩장 상단 또는 본문에 배치되는 인사말 영역. 신랑/신부 측에서 하객에게 전하는 메시지를 표시한다.

### Basic 항목
| 요소 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 제목 | text | O | 인사말 섹션 제목 (예: "소중한 분들을 초대합니다") |
| 본문 | textarea | O | 인사말 내용 |

### Advanced 항목
| 요소 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 폰트 스타일 | select | X | 본문에 적용할 폰트 선택 |
| 텍스트 정렬 | select | X | 좌측 / 중앙 / 우측 정렬 |
| 배경색 | color | X | 인사말 영역 배경색 |

### 기본값 예시
```
제목: "소중한 분들을 초대합니다"
본문: "서로 다른 길을 걸어온 두 사람이
하나의 길을 함께 걷고자 합니다.
귀한 걸음 하시어 축복해 주시면
더없는 기쁨으로 간직하겠습니다."
텍스트 정렬: 중앙
```

## 신랑 정보 컴포넌트 `초기`

### 설명
신랑의 이름 및 부모(혼주) 정보를 표시한다.

### Basic 항목
| 요소 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 신랑 성명 | text | O | 신랑 이름 |
| 신랑 부 성명 | text | O | 신랑 아버지 이름 |
| 신랑 모 성명 | text | O | 신랑 어머니 이름 |

### Advanced 항목
| 요소 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 관계 표기 | select | X | "장남 / 차남 / 아들" 등 |
| 고인 표기 | checkbox | X | 부 또는 모가 고인인 경우 표기 |

## 예식 일시 컴포넌트 `초기`

### 설명
예식 날짜와 시간을 표시한다.

### Basic 항목
| 요소 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 예식 날짜 | date | O | 예식 일자 (요일 자동 표기) |
| 예식 시간 | time | O | 예식 시각 (예: 오후 1시) |

### Advanced 항목
| 요소 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 달력 표시 | checkbox | X | 해당 월 달력을 함께 표시할지 여부 |

## 예식 장소 컴포넌트 `초기`

### 설명
예식장 정보와 위치를 표시한다.

### Basic 항목
| 요소 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 예식장명 | text | O | 예식장 이름 (예: "더채플 앳 청담") |
| 주소 | text | O | 예식장 도로명 주소 |

### Advanced 항목
| 요소 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 홀명 | text | X | 예식 홀 이름 (예: "3층 그랜드홀") |
| 지도 표시 | checkbox | X | 카카오맵/네이버맵 지도 노출 여부 |
| 교통 안내 | textarea | X | 대중교통, 주차 등 오시는 길 안내 |

## 연락처 컴포넌트 `초기`

### 설명
신랑/신부 및 혼주의 연락처를 표시한다. 전화 걸기 버튼 제공.

### Basic 항목
| 요소 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 신랑 이름 | text | O | 신랑 이름 (라벨용) |
| 신랑 연락처 | tel | O | 신랑 전화번호 |
| 신부 이름 | text | O | 신부 이름 (라벨용) |
| 신부 연락처 | tel | O | 신부 전화번호 |

### Advanced 항목
| 요소 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 신랑 부 이름 | text | X | 신랑 아버지 이름 |
| 신랑 부 연락처 | tel | X | 신랑 아버지 전화번호 |
| 신랑 모 이름 | text | X | 신랑 어머니 이름 |
| 신랑 모 연락처 | tel | X | 신랑 어머니 전화번호 |
| 신부 부 이름 | text | X | 신부 아버지 이름 |
| 신부 부 연락처 | tel | X | 신부 아버지 전화번호 |
| 신부 모 이름 | text | X | 신부 어머니 이름 |
| 신부 모 연락처 | tel | X | 신부 어머니 전화번호 |
